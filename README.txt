Este proyecto fue diseñado y desarrollado para el curso "Yo Puedo Programar" que brindó Junior Achievement Argentina (JAA) junto a Microsoft, en octubre-noviembre del año 2022.
Las herramientas utilizadas son HTML5, CSS3, JavaScript y Bootstrap.

Diseñado por Carlos José Castro Galante (@carlosjcastrog | @carlosjcastro | @charlescito en 2022.
